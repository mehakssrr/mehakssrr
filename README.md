### 🚀 Featured Projects

#### 🤖 [Multi-Agent Customer Support Assistant](https://github.com/mehakssrr)
A multi-agent AI system that automates enterprise-level customer support workflows. Specialized agents handle query intake, escalation, and resolution independently, demonstrating agent coordination and automated decision-making. Built under the **Enterprise Agents track**.

#### ✈️ [AI Travel Planner — TravelX](https://github.com/mehakssrr)
A space-themed, student-centric travel planning web app built during my IBM Skills Build internship. Features a dynamic budget calculator that breaks down costs across accommodation, food, transport, and activities based on destination, trip duration, and travel style. Architected in Python to stay maintainable and extendable without touching raw HTML.

#### 📈 [Bitcoin Price Direction Predictor](https://github.com/mehakssrr)
An end-to-end ML classification pipeline predicting next-day Bitcoin price direction from historical OHLCV data. Covers EDA, feature engineering (open-close spread, low-high range, quarter-end seasonality indicator), model training, and confusion-matrix validation. **XGBoost achieved the strongest performance.**  
`Python` `Pandas` `XGBoost` `Scikit-learn`

#### 🔍 [Hallucination Detector in AI Answers](https://github.com/mehakssrr)
An offline, no-API-key tool that detects hallucinations in AI-generated answers using a local RoBERTa-based NLI model. Frames hallucination detection as an entailment-vs-contradiction problem, outputting structured JSON with confidence scores for easy pipeline integration.  
`Python` `PyTorch` `Transformers` `NLI`

#### 🔊 [Text-to-Speech Optimization](https://github.com/mehakssrr)
A lightweight FastAPI web app that converts text to speech and streams real-time MP3 audio with low latency, using gTTS and Uvicorn. Built with a modular structure to support future multi-language and custom voice features.  
`Python` `FastAPI` `gTTS` `Uvicorn`

#### 🌐 [Language Translator](https://github.com/mehakssrr)
A Python-based text translation tool focused on simple, reliable I/O handling, structured for easy extension to additional language pairs.  
`Python`
